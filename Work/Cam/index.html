Cam World is a web application utalzing webcams.travel API and Madbox to create a easily accessble webpage where viewers can access the largest the Landscape webcam data base by country, region, or sepcific locations.

By clicking on anylocation on the map, it will return total number of live landscape webcams within the 20km radius and top 10 image feed. Users can search places or interacting with the map to find the area they want to check.

This project is designed to be very simple and did not catergorize the type of cameras on purpose. It gives the viewers the freedom to interperate why at certain area there are more cameras, and why certain type of cameras are located at certain areas as well.
