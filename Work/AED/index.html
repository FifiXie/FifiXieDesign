This At-Home AED Aid is a system created to for a collaboration project with Physio Control. The goal of this project is to design a system that can help be implimented for AED usage at home. Statisically, majority of cardiac arrest happens at patients home. However, there is only small percemtage of people have AED at home. This project is designed for people who are at high risks and would need a system that can help them at home.
